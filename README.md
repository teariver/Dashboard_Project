# Dashboard_Project

| Project | Description |
|---|---|
| [Advertising Campaign Performance Analysis](Advertising_Market_Analysis) | Power BI dashboard project for analyzing advertising campaign performance across multiple channels such as Google Ads, Facebook Ads, and LinkedIn. Includes KPI monitoring, conversion analysis, engagement tracking, star schema data modeling, and DAX-based business intelligence reporting. |
| [Customer Behavior Analytics Dashboard](Customer_Behavior) | Customer analytics and segmentation dashboard for the food delivery industry using RFM analysis, Power BI dashboards, DAX calculations, and customer behavior insights to support personalized marketing strategies and campaign optimization. |
| [Insurance Premium Prediction](Insurance_prediction) | Machine learning project using XGBoost Regressor to predict insurance premiums based on customer demographics, financial behavior, and health-related features. Includes Power BI dashboards, star schema data modeling, and predictive analytics visualization. |
| [Medical Treatment Compliance & Patient Analytics Dashboard](Medical_Regimen) | Healthcare analytics dashboard for monitoring treatment protocol compliance, ICD disease distribution, medication usage, and patient examination activities. Developed using Power BI, DAX, Power Query, and healthcare operational analytics. |
